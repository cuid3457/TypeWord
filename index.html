자주 묻는 질문

처음 쓰는 분이 가장 많이 물어보는 것들.

MoaVoca는 사전인가요?
아닙니다. MoaVoca는 학습 도구입니다. 단어를 한 번 찾아보고 끝내는 사전이 아니라, 단어를 저장해 두고 잊을 때쯤 다시 보여줘서 진짜 기억에 남도록 돕는 앱입니다. AI 정의는 학습 과정에서 의미를 빠르게 잡기 위한 보조 기능입니다.
어떤 언어를 지원하나요?
현재 영어, 한국어, 일본어, 중국어(간체), 스페인어, 프랑스어, 독일어, 이탈리아어 8개 언어를 지원합니다. 모국어와 학습 목표어를 자유롭게 조합할 수 있습니다 (예: 한국어 → 일본어, 영어 → 스페인어).
무료로 어디까지 쓸 수 있나요?
단어 저장 개수에 제한이 없고, 간격 반복 복습·8개 언어 원어민 발음·시험 큐레이션 단어장·친구·커뮤니티 기능을 모두 무료로 쓸 수 있습니다. AI 정의 호출 횟수와 사진 OCR 횟수만 일일/월간 제한이 있고, 광고가 포함됩니다.
오프라인에서도 쓸 수 있나요?
이미 다운로드한 단어장의 복습은 오프라인에서도 가능합니다. 단, 새 단어의 AI 정의·발음 생성·OCR은 서버 통신이 필요하므로 인터넷 연결이 있어야 합니다.
발음은 진짜 원어민인가요?
Microsoft Azure Neural Voice를 사용합니다. 8개 언어 각각에 대해 남녀 원어민 음성을 제공하며, 자연스러운 억양과 호흡까지 재현됩니다. 자동 재생 속도는 언어별로 보정해 일관된 학습 경험을 보장합니다.
시험 대비에 효과가 있나요?
TOEIC·JLPT·HSK·TOPIK 등 주요 어학 시험의 출제 빈도 데이터를 기반으로 큐레이션된 단어장을 무료로 제공합니다. 간격 반복 알고리즘이 잊을 때쯤 다시 보여주므로, 단기 암기가 아닌 장기 기억으로 정착시키는 데 적합합니다.
광고는 어떻게 작동하나요?
무료 사용자에게는 화면 하단 배너 광고와 일부 보상형 광고(보면 추가 학습 기회 획득)가 표시됩니다. 학습 중인 카드 위에 끼어드는 전면 광고는 없습니다. Premium 구독 시 모든 광고가 사라집니다.
어린이도 쓸 수 있나요?
앱 등급은 청소년·성인 학습자에 맞춰 설계됐습니다. 차별·혐오 표현은 단어 풀에서 제외하지만, 일반적인 어휘는 학습 가치를 위해 노출됩니다. 초등학생용 어휘 학습이 필요한 경우 다른 전용 앱을 권장합니다.
데이터는 어떻게 보관되나요?
학습 진도·단어장은 Supabase 클라우드에 안전하게 보관됩니다. 비밀번호는 해시 처리되어 운영자도 볼 수 없으며, OCR 원본 이미지는 처리 후 즉시 폐기됩니다. 자세한 내용은 개인정보처리방침을 참조하세요.
계정 삭제는 어떻게 하나요?
앱 내 설정 → 계정 → 계정 삭제로 즉시 가능합니다. 삭제 시 단어장·학습 기록·친구 관계 등 모든 개인 데이터가 30일 이내에 영구 삭제됩니다.
iOS / Android 앱은 언제 나오나요?
현재 App Store와 Google Play에 출시 준비 중입니다. 그동안에는 웹 앱(moavoca.com/app)으로 동일한 기능을 모두 이용할 수 있습니다.
구독은 어떻게 취소하나요?
App Store 또는 Google Play 결제 설정에서 언제든지 취소할 수 있습니다. 취소 후에도 결제 기간이 끝날 때까지는 Premium 기능을 계속 사용할 수 있습니다.
